sportbet:events-all

Get all events in the system.

sportbet:events-{event}

Get given event.

sportbet:members-all

Get all members in the event.

sportbet:members-{member}

Get given member.

sportbet:add-member

Add new member to the event.

sportbet:games-all

Get all games in the event.

sportbet:game-{game}

Get given game.

sportbet:add-game

Add new game to the event.

sportbet:bets-all

Get all bets in the event.

sportbet:bets-{member}

Get member bets.

sportbet:bets-{game}

Get game bets.

sportbet:add-bet

Add or update bet for a member and game.

sportbet:status-all

Get status (ranking) of the betting by listing all members ordered by their total betting points.

sportbet:status-{member}

Get detailed betting status for the member (showing points from individual game bets).

sportbet:edit

Edit current item.

sportbet:delete

Delete current item.
